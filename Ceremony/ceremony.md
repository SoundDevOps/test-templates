## 4. Generate SSH keys

### MacOS X
1. Open "Finder", choose Utilities from the "Go" menu.
2. Find the "Terminal.app" in the "Utilities" window.
3. Double-click to open the "Terminal.app".
4. Enter the following command in the Terminal window.
```
ssh-keygen -t rsa
```
